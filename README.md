# RCNL
我创建这个仓库是因为实在受不了那些作弊者赢了游戏还诋毁其他玩家，而且除了盯着屏幕发呆之外也无事可做。这个仓库的主要目标是提供准确的信息和数据，以便更轻松地进行封禁、​​检查和验证。

您可以在“讨论”标签页提问，或在“问题”标签页提交数据。

如果您想自行上传信息，可以使用“拉取请求”功能。不过，您的信息仍然需要经过我的确认才能被添加进去。
